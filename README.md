# Premier Schools Exhibition

A responsive frontend project built as a Frontend Developer assignment.

## Features

- Pixel-perfect UI from design reference
- Semantic HTML5 with BEM CSS methodology
- WCAG 2.2 AA accessibility compliant
- Fully responsive — mobile, tablet, desktop
- Cross-browser compatible (Chrome, Firefox, Safari, Edge)
- Smooth CSS animations — gallery scroll, hover effects
- No CSS frameworks — pure custom CSS only

## Tech Stack

- React 19
- Vite
- Pure CSS (BEM)

## Getting Started

```bash
npm install
npm run dev
```
